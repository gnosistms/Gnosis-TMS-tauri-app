# Modal Message Clarity Plan

## Goal

Replace modal supporting text that exposes implementation details or unclear internal
workflow language with user-facing copy.

## Scope

- Update only the modal messages identified in the copy audit.
- Preserve modal behavior, actions, state handling, and layout.
- Keep glossary and QA old-layout discard copy aligned through the shared modal.

## Verification

- Search for removed implementation-facing phrases after editing.
- Run a syntax check on touched JavaScript files if practical.
